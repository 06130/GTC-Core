# GTC Core 项目背景信息

## 项目概述
GTC Core 是一个基于 Minecraft 1.12.2 的模组项目，使用 CleanroomMC 构建系统进行开发。项目采用现代化的模组开发工具链，支持高级功能如 Mixin、Access Transformer 等。

## 技术栈
- **游戏版本**: Minecraft 1.12.2
- **构建系统**: Gradle (CleanroomMC)
- **开发语言**: Java 8
- **模组加载器**: Minecraft Forge
- **映射版本**: Stable 39

## 项目状态
当前项目处于早期开发阶段，包含基础框架结构：
- 主模组类 `GTC_Core.java`
- 标准配置文件结构
- 完整的构建和发布配置

## 开发环境特性
- 支持现代 Java 语法（通过 Jabel）
- 集成 Mixin 支持
- 完整的测试框架
- 多平台发布支持（CurseForge、Modrinth）
- 资源处理和变量替换机制

## 依赖关系
项目与 Cyberware 模组有依赖关系。

## 开发目标
构建一个功能完善的 Minecraft 模组，为后续功能扩展提供稳定的核心框架。
