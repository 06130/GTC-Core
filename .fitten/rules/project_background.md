# GTC-Core 项目背景信息

## 项目概述
GTC-Core 是一个基于 Minecraft 1.12.2 的 Forge 模组，专注于添加义肢和机械部件相关的游戏内容。

## 技术栈
- Minecraft 1.12.2
- Forge Mod Loader
- Cyberware API (用于义肢功能)
- Java 开发

## 主要功能
- 添加自定义义肢物品
- 实现机械部件的功能
- 与 Cyberware 模组的集成

## 项目结构
- 核心类: GTC_Core.java

## 开发环境
- Java 项目
- 使用 Gradle 构建
- 依赖管理通过 build.gradle 配置
